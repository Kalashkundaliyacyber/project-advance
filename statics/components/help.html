How to Scan

  1. Enter a target IP address or hostname in the Target field (e.g. 192.168.1.10)
  2. Select a Scan Type from the dropdown
  3. Click ▶ Scan to start
  4. Watch the progress bar — results appear automatically when done
💡 Use Service Detection for your first scan — it finds open ports and identifies what's running.
📊

How to Read Results

After a scan, results appear in the right panel tabs:

  • Risk — ports scored 0–10 by severity
  • CVE — known vulnerabilities matched to services
  • Findings — all open ports with version info
  • Next Steps — recommended follow-up scans
  • AI — AI-generated plain-English analysis
🔴 Critical / 🟠 High → act immediately  |  🔵 Medium / 🟢 Low → lower priority
🛡

What is a CVE?

CVE stands for Common Vulnerabilities and Exposures. Each CVE is a unique identifier (e.g. CVE-2023-38408) for a publicly known security vulnerability.

  • CVSS Score — severity rating from 0 to 10
  • Description — what the vulnerability does
  • Patch — how to fix it (upgrade, config change, etc.)
💡 ScanWise AI only shows CVE descriptions and patch advice — never exploit instructions.
📈

How to Use Graphs

Click 📊 Dashboard in the nav bar to see two graph views:

  • Graph (Radial) — Chart.js charts showing risk distribution, CVE severity, services discovered
  • Graph (OSINT) — Interactive D3 horizontal tree. Root = Host, branches = Ports → CVEs → Details

In the OSINT graph:

  • Click a node to expand/collapse its children
  • Scroll/pinch to zoom in and out
  • Drag to pan the view
  • Use ⊞ Expand and ⊟ Collapse buttons to control all nodes
💬

How to Use Chat

The chat assistant is on the left side of the Scan page:

  • Type any question about scanning, CVEs, or results
  • Use the quick question buttons for common queries
  • Click ☰ (top-left menu) → New Chat to start a fresh conversation
💡 The chat remembers context within a session. Start a New Chat when switching to a different topic.
⏹

Stopping a Scan

While a scan is running, the ⏹ Stop button appears next to Scan:

  1. Click ⏹ Stop
  2. Confirm in the popup
  3. The scan terminates safely — partial results are preserved
📄

Exporting Reports

Click ⬇ Report after a scan completes:

  • HTML — opens in browser, use Print → Save as PDF
  • PDF — direct PDF (requires pip install reportlab)
